# Kanzlei Landing Page

Landing page für eine Rechtsanwaltskanzlei.

## Stack

- **Next.js 16** (App Router)
- **React 19**
- **TypeScript**
- **Tailwind CSS v4**
- **Framer Motion**

## Lokale Entwicklung

```bash
npm run dev
```

Dann unter [http://localhost:3000](http://localhost:3000) öffnen.

## Sections

- **Navbar** – Fixed, responsive, mit mobilem Menü
- **Hero** – Hauptüberschrift, Trust-Indikatoren, CTAs
- **Leistungen** – 6 Rechtsgebiete als Karten-Grid
- **Über uns** – Kanzleiprofil & Werte
- **Team** – Anwaltsprofile
- **Kontakt** – Kontaktformular + Adresse
